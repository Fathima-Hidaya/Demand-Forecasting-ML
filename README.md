# Demand-Forecasting-ML
Machine learning project to forecast product demand using historical sales data.
This project predicts future product demand using historical sales data. It helps businesses plan inventory and reduce stock shortages or overstocking.

Technologies Used:
- Python
- Pandas & NumPy 
- Scikit-learn 
- XGBoost 
- Matplotlib

  Workflow
1. Data preprocessing
2. Feature engineering (date-based features)
3. Model training using XGBoost
4. Demand prediction

 Key Idea
The model learns patterns from past sales data to estimate future demand, supporting better inventory planning decisions.
